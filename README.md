# Sprout, Family Finance Platform

A single-file, interactive **concept demo** of a white-label B2B family-banking toolkit: the kind of product a bank could license to launch a GoHenry / Greenlight-style kids' debit-card experience in months instead of years.

Built as a product & front-end design portfolio piece. 

👉 **[Check out the demo](https://sprout-family-finance-platform.vercel.app)**.

## What's inside

**Bank admin portal**
- Portfolio overview: animated KPIs, account-growth line chart, transaction-volume breakdown
- Spending-rules engine: MCC category controls, limits, velocity rules
- White-label branding: live phone preview that updates as you type and recolour
- Implementation tracker: staged go-live timeline

**Customer experience**
- Parent app preview: interactive phone (freeze the card, send pocket money, spend donut)
- Junior app preview: a gamified kids' app with savings goals (progress rings), chore-to-earn tasks, a money-missions quiz, XP and badges

Plus light/dark mode, responsive layout, and count-up / chart entry animations.

## Run it

Just check out the Vercel deployed demo or run `index.html` in any modern browser. Chart.js and the icon set are inlined, so the app works **fully offline**.

## Tech

Vanilla HTML / CSS / JavaScript. [Chart.js](https://www.chartjs.org/) and [Lucide](https://lucide.dev/).
